    Introduction

    Welcome to the SendIT API! You can use our API to access SendIT API endpoints; which can retrieve, update or add parcel delivery order(s) in our database.

    SendIT is a courier service that helps users deliver parcels to different destinations. SendIT provides courier quotes based on weight categories.

    Authentication

    Sign Up

    HTTP/1.1 200 OK
Content-Type: application/json

{
  "email": "dapo@example.com",
  "password": "mango"
}

    This endpoint saves the user in the database, encrypting the password. It also creates a token using JWT, to authenticate the user and for privileges across the application.

    HTTP Request

    POST https://travissend-it.herokuapp.com/api/v1/auth/signup

    Sign In

    HTTP/1.1 200 OK
Content-Type: application/json

{
  "email": "dapo@example.com",
  "password": "mango"
}

    This endpoint validates that user exists in the database, encrypting the password. It also creates a token using JWT, to authenticate the user and for privileges across the application.

    HTTP Request

    POST https://travissend-it.herokuapp.com/api/v1/auth/login

    Get A Specific Parcel

    HTTP/1.1 200 OK
Content-Type: application/json

{
  "parcel_id": 1,
  "sender_id": 1,
  "parcel_kg": "16",
  "sender_address": "Herbert Macaulay Rd, Lagos",
  "recipient_address": "Agric Road, Ikorodu, Lagos",
  "status": "Delivered",
  "present_location": "Herbert Macaulay Rd, Lagos",
  "recipient_phone": "08099999999",
  "created": "11/22/2018"
}

    This endpoint retrieves a parcel with that :id in the database. It also checks a token using JWT, to authenticate the user and for privileges across the application.

    HTTP Request

    GET https://travissend-it.herokuapp.com/api/v1/parcels/:id

    URL Parameters

    Parameter Description
    ID The ID of the Parcel (which is currently "1")

    Errors

    The SendIT API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request is invalid.
    401 Unauthorized -- Your JWT authentication failed.
    403 Forbidden -- Your access was denied.
    404 Not Found -- Your Parcel(s) could not be found.
